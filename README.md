# 🏠 House Barbearia - Website

## Descrição
**House Barbearia** é um website responsivo para uma barbearia moderna, desenvolvido com **React** e **Tailwind CSS**.  
O site permite que os usuários vejam os serviços oferecidos, selecionem horários e agendem cortes diretamente pelo site, proporcionando uma experiência prática e profissional.

O projeto foca em **design clean, navegação intuitiva e responsividade**, garantindo uma boa experiência tanto em desktop quanto em dispositivos móveis.

---

## 🔹 Funcionalidades
- **Seções principais:**
  - Home com hero impactante
  - Serviços com preços e descrições detalhadas
  - Localização com mapa embutido
  - Formulário de agendamento interativo
- **Agendamento dinâmico:**
  - Usuário seleciona nome, horário e serviços desejados
  - Total do valor calculado automaticamente
  - Confirmação de agendamento com mensagem
- **Design responsivo:**
  - Layout ajustável para desktop, tablet e mobile
  - Botões interativos e dropdowns modernos
- **Navegação fácil:**
  - Menu fixo com links para as seções
  - Botão de agendamento sempre acessível

---

## 💻 Tecnologias Usadas
- [React](https://reactjs.org/) – Biblioteca para construção de interfaces
- [Tailwind CSS](https://tailwindcss.com/) – Framework de utilitários CSS para estilos rápidos
- [React Icons](https://react-icons.github.io/react-icons/) – Ícones modernos e responsivos
- [Netlify](https://www.netlify.com/) – Hospedagem e deploy do site
